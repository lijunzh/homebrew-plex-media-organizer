# Homebrew Tap for plex-media-organizer

This is the official [Homebrew](https://brew.sh) tap for [plex-media-organizer](https://github.com/lijunzh/plex-media-organizer).

## Install

```bash
brew install lijunzh/plex-media-organizer/plex-org
```

Or tap first:

```bash
brew tap lijunzh/plex-media-organizer
brew install plex-org
```

## Update

```bash
brew upgrade plex-org
```

## About

The formula is automatically updated by GitHub Actions on every release.
